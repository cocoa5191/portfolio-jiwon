Nakseo

Workshop by

´Gunhyuk Choi

2023 AI, Stable diffusion

"Nakseo" is a Korean term that means a hand-drawn sketch. This project involved creating abstract images drawn by hand during a Workshop with Gunhyuk Choi, using AI - Stable Diffusion's image masking feature
to generate the final graphics.

I used the SDXLImg2ImgPipeline APIs from Stable Diffusion to convert hand-drawn sketches into graphics using prompts.
